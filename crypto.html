Message Encryption
PropertyValueRationale
Cipher ChaCha20-Poly1305 AEAD construction — used in TLS 1.3, Signal, WireGuard. Resistant to timing attacks, no padding oracles, constant-time.
Key size 256-bit Maximum security margin. Post-quantum safe key size per NIST guidance.
Nonce 96-bit · random Cryptographically random per message — nonce reuse is impossible.
Auth tag 128-bit · Poly1305 Guarantees ciphertext integrity and authenticity. Tampered messages are rejected.
At-rest storage nonce + ciphertext Plaintext is never stored. Only encrypted blobs persist in memory.
Key Exchange
PropertyValueRationale
Algorithm X25519 ECDH Curve25519 — designed by D.J. Bernstein for resistance to side-channel attacks. Used in Signal, WireGuard, TLS 1.3.
KDF HKDF-SHA256 Derives the final ChaCha20 key from raw ECDH shared secret with domain separation.
Key scope per-room Compromise of one channel's key does not affect any other channel.
Authentication
PropertyValueRationale
Algorithm Argon2id PHC winner 2015. Hybrid of Argon2i (side-channel resistant) + Argon2d (GPU resistant). OWASP recommended.
Memory cost 64 MB High memory requirement defeats ASIC/FPGA brute-force hardware.
Time cost 3 iterations Each verification takes ~300ms — makes brute force economically infeasible.
Session cookie HttpOnly · Secure · SameSite=Strict Cookie inaccessible to any script. CSRF-resistant. Encrypted in transit only.
